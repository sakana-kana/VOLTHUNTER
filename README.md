# VOLTHUNTER

## ゲーム概要
攻撃を**ジャストで回避し、即座に切り返す**ことにフォーカスした  
電脳世界を舞台にした切り返し爽快アクションゲームです。

## 担当箇所
- ゲーム全体の**設計全般**
- **プレイヤー操作以外のシステム全般**を実装  
  - 敵AI・思考ルーチン  
  - 戦闘システム  
  - 各種ゲーム進行管理  
  - バランス調整を想定した仕組み設計

---

## 制作時に目指したこと・意識したこと
- プレイヤーに**回避行動を自然に促す**ゲームデザイン
- 回避成功時に得られる**爽快感の最大化**
- バランス調整が頻発することを前提とした、  
  **修正・調整しやすい設計**

特に、回避を成立させるためには  
**敵の思考ルーチンの設計が重要**だと考えました。

---

## そのために行ったこと

### ① 調整・確認しやすい設計
- プランナー側が数値や挙動を**調整・確認しやすい構造**を意識
- 実装側都合ではなく、  
  **調整する側の視点**を重視したシステム設計を行いました

### ② 敵AI・行動パターンの設計
- 敵の思考ルーチンや行動パターンを  
  **処理の流れが一目で分かる構造**で設計
- 複雑にしすぎず、  
  **意図した回避タイミングを作りやすい行動遷移**を重視しました

---

## 操作方法（コントローラー）

本作は **ゲームコントローラーでの操作を想定** しています。
（※Xboxコントローラー準拠）

- 移動：左スティック
- 視点移動：右スティック
- ジャンプ：Aボタン
- 回避：Bボタン
- 弱攻撃：Xボタン
- タメ攻撃：Yボタン
- ダッシュ：RBボタン
- スキル：RTボタン
- 電線移動：LBボタン

---
